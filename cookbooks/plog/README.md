# plog-cookbook

This cookbook install mysql,hhvm,nginx,node,npm,ghost.

## Supported Platforms

ubuntu 14.04

### plog::default

Include `plog` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[plog::default]"
  ]
}
```
